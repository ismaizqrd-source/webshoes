Ajustes

Configuración de pedidos y descuentos (necesitas hacer esto una vez en Cloudflare)

Cómo activar Pedidos y Descuentos (una sola vez)

  1. Cloudflare → tu proyecto de Pages → Settings → Functions → KV namespace bindings → crea/añade un namespace, llámalo ORDERS_KV
  2. Elige una contraseña tú mismo y añádela en Settings → Environment variables como secret ADMIN_KEY
  3. Stripe Dashboard → Desarrolladores → Webhooks → Añadir endpoint → URL: https://tudominio.com/api/webhook → evento: checkout.session.completed
  4. Copia el "Signing secret" que te da Stripe (empieza por whsec_...) y ponlo en Cloudflare como secret STRIPE_WEBHOOK_SECRET
  5. Rellena los dos campos de arriba en esta pantalla y dale a Guardar
